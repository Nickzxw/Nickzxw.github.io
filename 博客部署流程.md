## 每次发布新文章

### 1. 写文章

在 `hexo-source/source/_posts/` 目录下创建 Markdown 文件。

### 2. 生成并部署

```bash
cd /Users/a1-6/Documents/MyProject/hexo-source
npx hexo generate && npx hexo deploy
```

或手动执行:

```bash
cd /Users/a1-6/Documents/MyProject/hexo-source
npx hexo generate
# 然后将 public/ 下的内容复制到 GitHub 仓库
```
